# Customer Manager

## Technologies

- Backend:
  - Java
  - Spring Boot
- Frontend:
  - React
  - TypeScript

## Functionality

- Backend:
  - [x] Setup database (PostgreSQL)
  - [x] AddCustomer Route
  - [x] RemoveCustomer Route
  - [x] EditCustomer Route
- Frontend:
  - [x] AddCustomer Form
  - [ ] Customer List
  - [ ] RemoveCustomer Button
  - [ ] EditCustomer Button

The backend structure is based on a [YouTube tutorial](https://youtu.be/-mwpoE0x0JQ)
